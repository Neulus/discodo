# 디스코도 문서에 오신것을 환영합니다!

디스코도는 강화된 디스코드 오디오 플레이어입니다.

## 기능
- 독립적인 오디오 노드
- Youtube 추천영상 자동재생
- 크로스페이드 및 오디오 효과
- 싱크 Youtube 영상 자막

## 호환되는 소스

+ [youtube_dl](https://github.com/ytdl-org/youtube-dl)로 추출 가능한 모든 서비스
+ [libav](https://libav.org/)로 디묵싱 가능한 모든 포멧
